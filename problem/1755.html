얘도 숫자에 따른 문자열 매핑만 하면 끝이라 별거 없다.

정렬같은건 한번 해봤다면 라이브러리 쓰면 된다.


#include <cstdio>
#include <iostream>
#include <vector>
#include <algorithm>
char rd[10][16]={"zero","one","two","three","four","five","six","seven","eight","nine"};
int n,m;
std::vector<std::pair<std::string,int> > dat;
int main(){
    scanf("%d%d",&n,&m);
    for(int i=n;i<=m;i++){
        std::string t=i/10?rd[i/10]:"";
        t+=rd[i%10];
        dat.push_back(std::make_pair(t,i));
    }
    std::sort(dat.begin(),dat.end());
    int c=0;
    for(auto& i : dat)
        printf("%d%s",i.second,++c%10?" ":"\n");
}

이번거는 볼만한게 별로 없다. 정렬도 편하게 하려고 string 썼다.
