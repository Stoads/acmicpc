요약하면 특정수 자연수 n과 k에 대해서, n을 자연수 a,a+k로 나누는걸 더 못할 때 까지 반복하면 된다. 특정 수를 분할하면서 처리해 나가니 분할정복으로 볼 수 있겠고 그 방법은 재귀적 호출, 각 수를 노드로 보면 DFS로 볼 수 있겠다. 아님말고


#include <cstdio>
int n,m;
int dfs(int k){
    if(k<=m||k%2!=m%2)return 1;
    return dfs(k/2-m/2)+dfs(k-k/2+m/2);
}
int main(){
    scanf("%d%d",&n,&m);
    printf("%d",dfs(n));
}

추가적인 점이라면 n=2a+k여야 하므로 k가 홀수면 n도 홀수, k가 짝수면 n도 짝수여야한다는 점을 이해해서 조건을 짜야한다는 점이다.
