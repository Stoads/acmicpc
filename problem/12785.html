경로의 수기도 하고 파스칼의 삼각형도 떠오르는 그런 문제. 경우의 수와 관련된 간단한 수학문제임. 근데 크기가 200까지 가면 수가 많이 커지니 나머지 연산을 하라는 부분이 있다.


#include <cstdio>
long long pascal[999][999];
int main(){
    for(int i=0;i<999;i++)
        pascal[i][0]=pascal[0][i]=1;
    int n,m,x,y;
    scanf("%d%d%d%d",&n,&m,&x,&y);
    for(int i=1;i<n;i++)
        for(int j=1;j<m;j++)
            pascal[i][j]=(pascal[i-1][j]+pascal[i][j-1])%1000007;
    printf("%lld\n",pascal[x-1][y-1]*pascal[n-x][m-y]%1000007);
}

귀찮아서 초기화는 그냥 999로 때렸다. OJ인데 맞기만하면 불필요한 처리해도 상관 없는게 본질이다. 정석적으로 한다면 i<n||i<m이 된다.
