배열(혹은 문자열)에서 이전것과 비교해서 다르면 1씩 깎으면 되는 문제인데, 처음 처리가 중요하다.

처음을 예외로 따로 처리할 수 있게지만 맨 앞에 가상의 'R'문자를 추가함으로써 일괄적으로 처리할 수 있다.


#include <cstdio>
int n,k;
char s[1234]={'R'};
int main(){
    scanf("%d%d",&n,&k);
    scanf("%s",s+1);
    for(char* c=s+1;k&&*c;c++)
        if(*c==c[-1])
            k--;
    printf("%d",k);
}

처음 예외 처리를 생각 못해서 한번 틀렸는데, 악랄하게 예시를 4개나 줘놓고 예시에 3 2 RRR 외에는 R로 시작하는 예시가 없다.
