슬라이딩과 1단점프를 우선 세고, 남은 점프 회수로 2단점프를 세면 된다. 아예 전부 개수를 센다음에 체력을 계산해도 되고, 각각 순회를 하면서 체력을 계산해도 된다.


#include <cstdio>
int N,J,S,H,K;
char buff[3][123];
int main(){
    scanf("%d%d%d%d%d",&N,&J,&S,&H,&K);
    for(int i=0;i<3;i++)
        scanf("%s",buff[i]);
    for(int i=0;i<N;i++)
        if(buff[0][i]=='v')
            S--;
    for(int i=0;i<N;i++)
        if(buff[2][i]=='^'&&buff[1][i]=='.')
            if(J>0)J--;
            else H-=K;
    for(int i=0;i<N;i++)
        if(buff[1][i]=='^')
            if(J>1)J-=2;
            else H-=K;
    if(S<0)H+=S*K;
    printf("%d",H<=0?-1:H);
}

어쩌면 개수를 세는 편이 반복을 1번만 할거라 더 좋은 방법일지도?
