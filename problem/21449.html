Solved.ac 3번째 마라톤 참여 4번째 문제

n값이 많이 크다 10^9인데, 이를 int 배열로 선언해버리면 4GB가 나와버린다.

하지만 m값은 1000으로, 실질적으로 배열로 선언해도 접근하는 값은 1000개 밖에 안된다는 얘기다. 이럴땐 map을 쓰면 된다.


#include <cstdio>
#include <map>
int n,m,k;
std::map<int, int> t;
int main(){
    scanf("%d%d",&n,&m);
    long long res=0;
    for(int i=1,x;i<=m;i++){
        scanf("%d",&x);
        res+=n-i+t[x];
        t[x]=i;
        printf("%lld\n",res);
    }
}

간단하게 그사람이 메시지를 보낼때 이전 시점에서 몇개의 메시지를 읽었는지만 체크하고, 나머지사람들은 다 안읽은걸로 늘려주면 끝인 문제다.
