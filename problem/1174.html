기초 백트래킹 문제. 백트래킹 조건이 너무 간단해서 그냥 조건주고 BFS로도 잘만 풀릴 거다. 작은 수부터 구하고 싶다면 BFS를 하면 되겠지만 상대적으로 구현이 편리한 DFS를 때리고 정렬해버리는게 더 편하다.


#include <cstdio>
#include <cstring>
#include <algorithm>
long long dec[1234567];
int dn;
void dfs(long long p,long long v,long long mul){
    for(long long i=p;i<10;i++)
        dfs(i+1,dec[dn++]=i*mul+v,mul*10);
}

int main(){
    memset(dec,0xff,sizeof(dec));
    dfs(0,0,1);
    std::sort(dec,dec+dn);
    int n;
    scanf("%d",&n);
    printf("%lld",dec[n-1]);
    
}

조건이 너무 간단해서 좋은 백트래킹 문제 예시는 못되는거 같다. 백트래킹을 익히고 싶다면 N-queen 문제를 찾아서 푸는 편이 도움이 될 것이다.
