N*M 안에서 가장 큰 정사각형의 크기를 묻는 문제인데, N==M인 경우만 조심하면 된다.

어쨌거나 결론은 N==M의 경우 (N-1)^2+1가 최대 넓이가 된다는 이야기다.


#include <cstdio>
int N,M;
int main(){
    scanf("%d%d",&N,&M);
    if(--N>--M)N^=M^=N^=M;//swap
    printf("%d",N==M?N*N-(N-1)*2:N*N);
}

N과 M의 대소관계를 일정하게 바꿔주면 굉장히 코드가 간단해지는게 팁이라면 팁
