백준 OJ(Acmicpc.net)의 처음 문제로, OJ의 사용법 학습을 목적으로 하는 간단한 문제이다. 이를 통해 필자는 해당 블로그의 내용을 확인하고, 독자는 백준 OJ를 어떻게 사용하는지에 대해 알아가는 시간을 가질 수 있을 것 같다.

우선 입력으로 두 수가 주어진다. 조건을 보면 각 수는 0과 10 사이의 자연수임을 알 수 있다.

따라서 두 수를 입력 받고, 그 수를 더한 값을 출력하기만 하면 되는 간단한 문제이다.

여기서 주의할 점은, 해당 조건 외의 입력은 주어지지 않고, 출력은 정해진 내용만 해야한다.(사용자가 알아보기 쉬워지는 추가적인 표현 같은 것을 넣으면 오답 처리 된다.) 필자는 처음 C언어를 공부할 때 이걸 몰라서 틀렸던 적이 있었다.

간단하지만 코드 영역 테스트를 위해 코드를 작성해보자



#include <cstdio>
int main(){
  int a,b;
  scanf("%d%d",&a,&b);
  printf("%d",a+b);
}

이에 대한 입력으로 다음 값을 주면


  3 5

다음과 같은 출력이 나올 것이다.


  8

다쓰고 결과를 보니까 생각보다 쓸만한거 같다.

문제가 있다면 코드 영역이 꺽쇄 같은 것들을 이런 변환 사이트를 쓰지 않으면 귀찮다는건데 감내해야지 뭐..
